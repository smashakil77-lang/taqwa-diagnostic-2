# TAQWA Manager React + Firebase

## Deploy
1. Upload this folder to GitHub.
2. Import repo in Vercel.
3. Framework: Vite. Build command: `npm run build`. Output: `dist`.
4. Firebase Authentication Email/Password must be enabled.
5. Firestore must be enabled.
